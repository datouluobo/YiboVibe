> 版本：1.0.0  
> 日期：2026-04-25  
> 状态：已实现

---

## 1. 设计目标

在与 YiboFlow 整体毛玻璃 + 蓝紫主题风格一致的前提下，解决以下问题：

| 问题类型 | 现状 | 目标 |
|---------|------|------|
| 布局 | 单列列表，空间利用率低 | 左列表 + 右预览的主从双栏布局 |
| 大图查看 | 图片被限制 120px 高度 | 右侧预览区全尺寸展示 |
| 文本详情 | 文本直接显示在卡片中，无法查看完整内容 | 预览区完整展示 + 支持滚动选择 |
| 剪贴板读取 | 仅被动监听，不稳定 | 被动监听 + 主动轮询双通道 |
| 复制功能 | 无 | 卡片内 + 预览区均可一键复制 |
| 历史管理 | 仅保留 20 条，无清除机制 | 保留 50 条 + 手动/自动清除 |

## 3. 筛选栏规格

三维度组合过滤，同时生效（AND 逻辑）。

**时间（单选）：** 全部 / 今天 / 昨天 / 近7天 / 近30天  
**类型（单选）：** 全部 / 文本 / 图片  
**方向（单选）：** 全部 / 发送 / 接收

样式：胶囊标签，选中 `var(--color-primary)`，未选中 `var(--color-text-muted)`。

---

## 4. 左侧活动流卡片

### 文本卡片
- 方向图标 + 状态文字 + 右对齐时间
- 最多 2 行预览，溢出截断
- 右下角：复制 + 删除图标按钮

### 图片卡片
- 方向图标 + 状态文字 + 时间
- 固定 80px 缩略图
- 右下角：复制 + 删除图标按钮

### 卡片交互
- 默认：`var(--color-surface-elevated)` 背景
- 选中：左边框 3px `var(--color-primary)`
- 入场动画：framer-motion opacity + y + scale

### 右下角图标按钮
- 复制（Copy 图标）：调用 `write_to_clipboard` 写入系统剪贴板
- 删除（Trash2 图标）：直接移除该条记录
- 图标 14px，hover 变亮

---

## 5. 右侧预览区

- 空状态：图标 + "选择一条记录查看详情"
- 文本：完整内容，等宽字体，可滚动，user-select: text
- 图片：全尺寸 object-fit: contain，自适应填满
- 元信息：时间 / 状态 / 类型
- 操作：复制 / 删除 / 保存（图片）

---

## 6. 底部工具栏

- 显示记录数量
- 手动清除：下拉菜单（3天前 / 7天前 / 30天前 / 全部清除）
- 自动清除：可选 1/3/7/14/30天/关闭，存 localStorage
- 组件挂载时自动清理过期记录

---

## 7. 剪贴板读取改进

双通道：被动监听 `clipboard-event` + 主动轮询 `read_clipboard_content` 每 2 秒兜底。

    imageHeight?: number;
