## 1. 项目愿景

**YiboFlow** 是一款以本地优先与私有同步为核心的跨平台生产力增强套件。它通过 Rust 编写的核心引擎，在保障隐私与安全的前提下，为用户提供输入增强、数据实时同步、多端协作与本机直连 API 测试能力。

## 3. 技术核心亮点

- **隐私至上 (Privacy First)**：
  - **E2EE 端到端加密**：所有同步数据在客户端加密，NAS 服务端仅作为“聋哑转运站”，无法解密用户内容。
  - **私有化部署**：支持 Docker 部署于个人 NAS，数据完全自主可控。
- **极致性能 (High Performance)**：
  - **Rust 核心引擎**：系统级 Hook 与数据处理层由 Rust 编写，确保低延迟 (<50ms) 与极低内存占用 (<80MB)。
  - **进程分离架构**：后台 Core 引擎持续运行，前端 UI 随用随开，互不干扰。
- **现代化体验 (Modern UX)**：
  - **全平台支持**：覆盖 Windows、NAS (Docker)，并规划 iOS/Android 移动端。
  - **丰富主题系统**：内置深渊毛玻璃 (Dark Glass)、午夜霓虹 (Neon) 等 5 套高审美主题。
  - **i18n 国际化**：完美支持中英文无双缝切换。

## 4. 特色子系统介绍

### 4.1 FlowSnap (锦囊) — 缩略语引擎
- **双替换策略**：支持“模拟按键”与“剪切板粘贴”两种方式，兼容普通编辑器与复杂富文本环境。
- **IME 兼容性**：自动检测输入法状态，避免在中文模式下产生冲突。

### 4.2 FlowProbe (探脉) — 本机直连测试工具
- **显式协议类型**：支持 OpenAI Compatible、Ollama、Gemini OpenAI Compatible、Anthropic、Custom。
- **直接测试**：所有请求由当前设备直连目标地址，服务端不代理第三方 AI API。
- **标准化结果**：输出结果码、解释摘要、认证状态、识别协议、延迟和模型数量。

### 4.3 FlowRules (中枢) — 应用专家
- **细粒度控制**：不再是简单的全局开关，而是可以针对每个具体程序（如 VS Code、CMD）精确开启或关闭特定功能。
